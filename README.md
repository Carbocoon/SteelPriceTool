# 使用须知

## 使用方法

点击exe文件运行，出现黑色方框，等待，会自动打开浏览器，即可使用。

## 常见问题

1. 打开浏览器以后没有加载？

   检查打开的浏览器是否是最古老的IE浏览器，如果是，将地址复制到你最常使用的浏览器中。
2. 生成的表格中有一些项目没有被识别？

   检查表头是否与附录中的对应，如果没有请改正
